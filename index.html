その過程で、農業や漁業といった一次産業の現場にも関わり、身体を動かす仕事と、仕組みを整える仕事の両方を経験しています。

一人で黙々と作業することが得意で、細部を確認しながら、時間をかけて丁寧に積み上げていく制作スタイルを大切にしています。 分からないことを曖昧にせず、調べ、試し、理解する。

この積み重ねを、制作や運用の現場においても重視してきました。 自然や散策、ものづくりが好きで、都市生活の中でも食・電気・DIYといった分野に関心を持ち、日々学びと実践を続けています。

Skills

HTML/CSS

HTML/CSSを用いた、保守性の高いコーディングが得意です。単にデザインを再現するだけでなく、Tailwind CSSを用いたユーティリティファーストな設計により、保守性と実装速度を両立したコーディングや、レスポンシブ対応を完遂できます。また、環境変数による設定管理やネットワーク層の知識を活かし、バックエンドとの連携を意識した実装が可能です。ログから問題を特定するデバッグ力を武器に、ブラウザ間の差異や表示崩れにも迅速かつ論理的に対処し、安定したUIを提供します。

JavaScript

JavaScriptを用いた動的なフロントエンド開発に加え、Node.js環境でのサーバーサイド実装も可能です。API連携時の非同期処理（Async/Await）やエラーハンドリングに精通しており、ETIMEDOUT等のログから原因を特定する深いデバッグ力を備えています。また、環境変数による機密情報の管理やTLS設定など、セキュリティとネットワークを意識した「堅牢なコード」が書けることが、エンジニアとしての最大の強みです。

Python

Pythonを用いた業務効率化やサーバーサイド開発を得意としています。 ライブラリを活用したデータ処理に加え、外部APIとの連携やスクレイピングの実装実績があります。 また、自動キャプチャ後にPDFを生成するアプリケーションを開発した経験があります。 ページ終端の自動検出やページ送りの検出ロジックを実装し、手作業を介さずに一連の処理を完結させる仕組みを構築しました。
